# Fever Bot — Vercel deployment

Static-only project. Hosts the public ICS feed and a one-screen landing page.

## Files
- `public/fever-2026.ics` — generated by `src/generate_ics.py`. Replaced by the daily refresh task.
- `public/index.html` — landing page with the feed URL + download button.
- `vercel.json` — sets `text/calendar` content-type and CORS headers on the ICS route.

## Deploy
This folder is deployed via the Vercel MCP `deploy_to_vercel` tool under team `mitchleonards-projects-c174884e`.

## Regenerating the ICS
```
python3 ../src/generate_ics.py
cp ../src/fever-2026.ics public/fever-2026.ics
```
